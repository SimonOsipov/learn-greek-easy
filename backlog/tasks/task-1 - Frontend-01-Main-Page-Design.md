## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Design and create mockups/wireframes for the main landing/dashboard page. This page serves as the central hub for authenticated users.

**Scope:**
- Define layout and information architecture
- Identify key UI components and placement
- Create wireframes/mockups (low/high fidelity)
- Establish visual hierarchy and user flow
- Ensure mobile-responsive design

**Key Decisions Made:**
- Primary: Dashboard showing learning progress and stats
- Top 3 metrics: Cards due today, Learning streak, Overall progress
- Simple list with progress bars for deck display
- Warm and encouraging visual design to reduce exam anxiety
- Consistent layout (no dynamic changes based on context)

**Deliverables:**
- High-fidelity HTML mockup (index.html)
- Wireframe specifications (01.02-wireframe-design.md)
- Component identification (23 components)
- Style Guide with complete design system
- Design decisions document (2,800+ words)
