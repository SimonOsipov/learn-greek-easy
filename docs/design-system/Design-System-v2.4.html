02 · Principles

Three rules.

01
Depth, not noise.

Surfaces float on translucency and shadow — never on borders alone. Glass is a structural element, not decoration.

02
Color earns its keep.

Saturated hues only carry meaning — primary action, success, danger, gradient CTAs. Everything else is calm tinted-neutral.

03
Motion as feedback.

Every transition tells the user what just happened. Bars grow, numbers tick, content lifts. Never decorative.

In motion

Motion.

Movement is functional, never decorative. Three durations, one curve, and a strict rule: animate what changed, not the whole frame.

Duration

Three speeds.

200ms for state · 300ms for entry · 500ms for big transitions. Anything longer feels broken.

Fast · 200ms
Hover · focus
Buttons, links, card lifts. Imperceptible if it's right.
Default · 300ms
Reveal · entry
Cards entering, accordions, popovers, toast slides.
Slow · 500ms
Page · view
Route changes, modal opens, theme transitions.
Easing

One curve.

cubic-bezier(0.4, 0, 0.2, 1) — Material's "smooth" curve, applied universally for predictability.

--ease-smooth

  0.2, 1
)
Starts decisively, settles softly. Avoids the "elastic" feel that breaks trust on a learning surface.

Rules

Non-negotiable.

Respect prefers-reduced-motion

All animations collapse to 0.01ms. The codebase already handles this — never override.

No bouncing chrome

Spring physics belong on flashcards (pop-in), not on navigation, modals, or buttons. Use the smooth curve.

Compose, never overlap

A single element animates one property at a time. If it scales and fades, the durations match.

Stagger only on lists

40ms between items, max 8 items, then everything appears at once. No infinite cascades.

In words

Voice & tone.

We're a friend who happens to know Greek — patient, warm, and honest about the work. Never a textbook. Never a cheerleader.

Voice is
  • ✓ Warm, not saccharine
  • ✓ Specific, not generic
  • ✓ Honest about difficulty
  • ✓ Curious about culture
  • ✓ Brief, then quiet
Voice isn't
  • ✗ Rah-rah motivational
  • ✗ Streak-shaming
  • ✗ Translingual punning
  • ✗ Emoji-heavy
  • ✗ Stiff or academic
Tone shifts
  • success · understated, not loud
  • error · plain, blameless
  • empty · inviting, never lonely
  • streak · proud but quiet
  • danger · clear consequence

03 · Reduced motion

Respect the setting.

When prefers-reduced-motion: reduce is on, all transitions collapse to 0.01ms. The streak still increments — it just doesn't bounce.

@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: .01ms !important;
    transition-duration: .01ms !important;
  }
}

06 · Screen reader

Speak plainly.

Live regions announce what changed, never the whole frame. Greek words carry lang="el"; transliterations carry lang="el-Latn". Every icon-only button has an aria-label.

Live region · streak
<div role="status" aria-live="polite">
  Streak: 12 days
</div>
Announced as "Streak: 12 days" only when the value changes — never on initial render.
Lang attributes
<p lang="el">καλημέρα</p>
<p lang="el-Latn">kaliméra</p>
Greek text uses Greek TTS. Transliteration uses Latin alphabet voice. No more robot-Greek pronouncing English words.
Icon-only button
<button aria-label="Hear pronunciation">
  <svg aria-hidden="true" />
</button>
SVG is decorative; the button carries the label.
Loading announcement
<span aria-live="polite" class="sr-only">
  Fetching today's queue…
</span>
Visually hidden, screen-reader-audible. aria-busy on the parent region.

08 · Ship list

Before merge.

Eight quick checks. Run them on every new screen — they take longer to skip than to do.

  • Tab through every interactive. Order matches reading order; focus is always visible.
  • Run a screen reader on the practice loop. Cards announce; grading announces; streak announces.
  • Toggle reduced-motion in OS. No transition longer than 0.01ms; content still legible.
  • Zoom to 200%. No horizontal scroll, no hidden controls, no clipped text.
  • Disable color (grayscale). State (selected, due, mastered) still legible from shape and label.
  • Test with auto-translate off. Greek text never translates to English; lang attributes set.
  • Click every icon-only button. Each one carries an aria-label.
  • Run axe-core in CI. Zero violations is the floor — every PR.
