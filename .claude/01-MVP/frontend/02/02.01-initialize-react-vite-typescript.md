## A. Task Overview

### Purpose
Initialize a modern React project using Vite as the build tool and TypeScript for type safety. This creates the foundation for the Greek Language Learning SaaS frontend application.

### Why Vite?
- **Lightning-fast HMR (Hot Module Replacement)**: Changes reflect instantly during development
- **Optimized production builds**: Automatic code splitting and tree-shaking
- **Modern ES modules**: Native browser support for faster development
- **Built-in TypeScript support**: Zero configuration needed
- **Smaller bundles**: Better optimization than traditional bundlers

### Why TypeScript?
- **Type safety**: Catch errors at compile time, not runtime
- **Better developer experience**: IntelliSense, auto-completion, and refactoring
- **Self-documenting code**: Types serve as inline documentation
- **Fewer runtime errors**: Reduces bugs in production
- **Industry standard**: Most modern React projects use TypeScript

### Estimated Time
- **Initial setup**: 10-15 minutes
- **Verification and cleanup**: 5-10 minutes
- **Total**: 20-30 minutes

## F. Troubleshooting

### Common Issues and Solutions

#### Issue 1: Port 5173 already in use
**Error**: `Error: Port 5173 is already in use`

**Solutions**:
```bash
# Option 1: Kill the process using port 5173
# On macOS/Linux:
lsof -i :5173
kill -9 <PID>

# On Windows:
netstat -ano | findstr :5173
taskkill /PID <PID> /F

# Option 2: Use a different port
npm run dev -- --port 3000
```

#### Issue 2: npm install fails
**Error**: `npm ERR! code EACCES` or dependency errors

**Solutions**:
```bash
# Clear npm cache
npm cache clean --force

# Delete node_modules and package-lock.json
rm -rf node_modules package-lock.json

# Reinstall
npm install

# If permission errors on macOS/Linux:
# DO NOT use sudo! Instead, fix npm permissions:
npm config set prefix ~/.npm-global
export PATH=~/.npm-global/bin:$PATH
```

#### Issue 3: TypeScript errors on startup
**Error**: TypeScript compilation errors in terminal

**Solutions**:
```bash
# Ensure TypeScript is installed
npm install --save-dev typescript

# Regenerate tsconfig
npx tsc --init

# Check Node types are installed
npm install --save-dev @types/node
```

#### Issue 4: Module not found errors
**Error**: `Cannot find module 'react'` or similar

**Solutions**:
```bash
# Ensure you're in the correct directory
pwd
# Should be: .../learn-greek-easy-frontend

# Reinstall dependencies
rm -rf node_modules package-lock.json
npm install

# Verify React is installed
npm list react
```

#### Issue 5: Vite command not found
**Error**: `sh: vite: command not found`

**Solutions**:
```bash
# Ensure you're in project directory
cd learn-greek-easy-frontend

# Run with npx
npx vite

# Or reinstall Vite
npm install --save-dev vite
```

#### Issue 6: Browser doesn't auto-open
**Note**: Vite doesn't auto-open browser by default

**Solution**:
```bash
# Manually open browser to http://localhost:5173

# Or modify package.json to auto-open:
"scripts": {
  "dev": "vite --open"
}
```

#### Issue 7: Hot reload not working
**Symptoms**: Changes don't reflect in browser

**Solutions**:
1. Check file is saved
2. Check terminal for compilation errors
3. Hard refresh browser: Ctrl+Shift+R (Cmd+Shift+R on Mac)
4. Restart dev server
5. Check that you're editing files in `src/` directory

#### Issue 8: ESLint errors
**Error**: Linting errors when running `npm run lint`

**Solutions**:
```bash
# Auto-fix fixable issues
npm run lint -- --fix

# Or temporarily disable in eslint.config.js
# (not recommended for production)
```
